 Host = 2^9  = 512 -2 510 (cumple con los 500)

  32- 9 == 23 bits para la red

[23 bits de red ]    [9 bits host]

  
----------------------------------------

Red original: 172.30.0.0/16 (máscara 255.255.0.0).

bits prestados 23 -16 = 7 bits
